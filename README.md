# DadsFam Cache

**Fast and friendly WordPress page caching plugin** with smart automatic clearing and Pro performance tools.

![DadsFam Cache](CACHE%20DADSFAM%20SCREENSHOT%201.png)

**Version:** 1.2.1  
**Author:** DadsFam  
**Website:** [dadsfam.co.za](https://www.dadsfam.co.za/plugins-dadsfam-co-za/)

---

### Features

#### ✅ Free Features
- Full-page disk caching (serves cached HTML before WordPress loads)
- Gzip compression + smart browser caching rules
- Automatic cache clearing on post edits, updates, etc.
- Built-in exclusions (cart, checkout, my-account, logged-in users)
- Ignores tracking parameters (utm_*, fbclid, etc.)
- One-Click Speed Setup
- Cache stats and manual clear/test tools

#### ⭐ Pro Features
- HTML, CSS & JS Minification
- Critical CSS + render-blocking elimination
- WebP image conversion (auto on upload)
- Lazy loading for images & iframes
- Delay / Defer JavaScript
- Font optimization & preloading
- Link prefetch on hover
- CDN URL rewriting
- Database cleanup (revisions, transients, spam)
- Advanced performance dashboard

---

### Installation

1. Download the latest release from this repository
2. Upload the `dadsfam-cache` folder to `/wp-content/plugins/`
3. Activate **DadsFam Cache** in WordPress
4. Go to **DF Cache** → Click **One-Click Speed Setup**

---

### Quick Start

1. Run **One-Click Speed Setup** after activation
2. Clear cache once after making changes to your site
3. (Optional) Enter your DadsFam license key to unlock Pro features
